Run 3

Endless Space Running Game

About Run 3

Run 3 is an exciting endless running game set in space. Navigate through tunnels floating in space, avoid holes, and keep running as long as you can. The unique gravity mechanics allow you to run on walls and ceilings!

Game Features

Space Tunnels

Run through mysterious tunnels floating in outer space with stunning visuals.

Gravity Defying

Run on walls and ceilings - gravity changes based on which surface you're on.

Multiple Characters

Unlock different characters with unique abilities as you progress.

Endless Challenge

Randomly generated levels mean endless gameplay and new challenges.

How to Play

  • Arrow Keys / WASD: Move left and right
  • Space / Up Arrow: Jump
  • R: Restart level
  • Tip: Jump on walls to change gravity direction

Avoid falling into holes! If you fall into space, you'll have to restart. Use the walls strategically to navigate around obstacles.

Tips for Success

Pay attention to the tunnel ahead and plan your jumps. Sometimes running on the walls or ceiling gives you a better path. Practice makes perfect - the more you play, the better you'll get at reading the tunnels!
